# **App Name**: Safe Sandbox

## Core Features:

- Code Input: A text area where users can input code to be executed.
- Resource Limits: Configurable CPU and memory limits to constrain code execution.
- Real-time Output: Display the output of the executed code in real time.
- Process Control: Allow users to manually terminate the code execution process.
- AI-Powered Threat Detection: Utilize an AI tool to analyze code for potential security threats before execution. The AI should determine whether to pause or terminate the code execution.

## Style Guidelines:

- Background color: Dark gray (#222222) for a modern, secure feel.
- Primary color: Subtle green (#98FF98) to indicate security and operational status.
- Accent color: Cyan (#00FFFF) for interactive elements and important notifications.
- Body and headline font: 'Inter' sans-serif for clear readability and a modern, neutral look.
- Code font: 'Source Code Pro' monospace for code display.
- Use security-themed icons to represent different features and states.
- Clearly separated sections for code input, output, and settings.
- Subtle animations for process status and feedback.
